FAQ

Is Unbinge free?
Yes, completely free. No account required.
Does it work on mobile?
Browser extensions work on desktop Chrome and Edge. On mobile, YouTube Kids is the safest option.
Can my child bypass it?
The extension can be removed from the browser settings page. For stricter control, use supervised Chrome profiles or enable managed browser policies.
How do channels get approved globally?
Any user can submit a channel request. Our team manually reviews it for age-appropriateness and educational value before adding it to the global list.
Does Unbinge collect personal data?
No. The extension stores all settings locally on the device. Channel requests only send the channel URL — no names, emails, or identifiers.
